# Job Portal UI

A simple and responsive **Job Portal UI** built using **HTML, CSS, and JavaScript**. This project allows users to search for jobs by title and location, view featured jobs, and navigate through login and registration pages.

## Features

- Responsive Home Page
- Job Search by Title and Location
- Featured Job Listings
- Job Categories
- Login Page
- Register Page
- Interactive Search using JavaScript
- Clean and User-Friendly Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
